# 🚀 Mohammad Nasim Safi - Portfolio

A stunning, modern portfolio website built with Next.js 14, featuring liquid glass design, GitHub integration, and a full CMS.

## ✨ Features

- 🎨 **Liquid Glass Design** - Modern glassmorphism with dynamic effects
- 🌓 **Dark/Light Mode** - Automatic theme switching
- 🔄 **GitHub Integration** - Auto-sync projects from GitHub
- 📱 **Fully Responsive** - Works on all devices
- ⚡ **Blazing Fast** - Built with Next.js 14 App Router
- 🎭 **Advanced Animations** - Framer Motion throughout
- 📊 **Admin Dashboard** - Full CMS for all content
- 🔐 **Authentication** - Secure admin access
- 📧 **Contact Form** - With database storage
- 🗄️ **PostgreSQL** - Neon DB with Drizzle ORM
- 🎯 **SEO Optimized** - Meta tags and semantic HTML

## 🛠️ Tech Stack

- **Frontend:** Next.js 14, React, TypeScript, Tailwind CSS, Framer Motion
- **Backend:** Next.js API Routes, NextAuth.js
- **Database:** PostgreSQL (Neon DB), Drizzle ORM
- **Deployment:** Vercel
- **Integrations:** GitHub API, Radix UI, Lucide Icons
